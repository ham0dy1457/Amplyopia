# WebApp Starter (HTML/CSS/JS)

A minimal, responsive starter using plain HTML, CSS, and vanilla JavaScript.

## Structure

```
webapp/
  index.html
  css/
    styles.css
  js/
    app.js
  assets/
```

## Getting started

- Open `index.html` directly in your browser, or
- Serve locally for better dev experience:

  - PowerShell: `python -m http.server 8000`
  - Node (if installed): `npx serve .` or `npx http-server .`

Then visit `http://localhost:8000`.

## Customize

- Colors and spacing are in `css/styles.css` (`:root` variables).
- Navigation and sections are in `index.html`.
- Interactivity lives in `js/app.js`.

## Deploy

Host the folder on any static host (GitHub Pages, Netlify, Vercel). Just upload the files.
